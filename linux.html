Linux is incredible, and Free!!

Linux utitlities to help

Linux is one of the most incredible pieces of technology there is, there is so much meaningful information that can be displayed through simple Linux commands. Below are some of my personal favorite Linux services.

Ukuu

The great thing about Ukuu is being able to update your kernel to any released version that is available. When you first start the computer up, enter the BIOS menu and you are able to select the kernel desired. If the box breaks than all you need to do is return to the BIOS menu and reselect the desired kernel. An interesting fact is the information provided with the kernel number. The current kernel is 5.4.0; if the second number is even then it signifys the kernel is stable. If the middle number is odd then the kernel is more bleeding edge(more bugs/beta testing).

IPTables (compareable to UFW)

Setting up your own firewall rules is crucial. I usually limit access to port 22 (SSH), port 80/8080 (HTTP/HTTPS), and port 20 (FTP). The utitiles that are most suspect are the file sharing services. Wget, curl, ftp, telnet are the known file sharing services, so limiting port access to these is a great start to securing your system. The learning curve for this is rather steep (compared to UFW/GUFW) but the rules are stable. Here is a good tutorial with good examples to get iptables working.

AppArmor

AppArmor is a preinstall service that allows for the auditing of profiles or services. This service allows for the configuration of kernel programs and the allocation of resources. By placing a speific service in "enforcement" mode will report any policy violation into syslogd. You have the ability to download blacklisted profiles in order to stregthen your system further. There are firefox profiles that will grant specific resources to the browser and will Allow or deny requests. I find this very useful, another great addition to apparmor is firejail. Firejail can load any service into its own sandbox environment in order to keep the resources separated.
