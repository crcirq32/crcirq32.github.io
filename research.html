Setup a Linux USB/CD bootable drive

    Items needed for Linux:
     * An internet Connection.
    * BelenaEtcher, UUSBI, DD (dd=if/dev/XImageOfDistro of=/dev/sdx).
    * Ubuntu || Mint || Fedora
    * USB/MicroSD device ~4gb+ (Dependent on Distro size).
    * Separate computer (One to put Linux on).


For this tutorial I will be using the software Etcher for flashing the USB drive. I have read that the dd command works wonders (dd=if/dev/XImageOfDistro of=/dev/sdx)


The photo above shows what the process should look like through the Etcher GUI (Graphical User Interface), there are only 3 steps. The first step requires you to locate the image (.iso, .img, iso.gz, img.zip). After selecting the desired image, insert a blank USB drive or CD into the computer. Etcher software should auto recognize the flashdrive. If not, click the "select button" and select the USB. After that, click flash. That's it.
