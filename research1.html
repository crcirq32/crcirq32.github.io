Research: Hosting a Website

Hosting vs Domain name

* In order to create a website, you need to purchase a Web Domain name (www.XYZ.com), as well as server space (hosting) in order for others to view your page.

* There are very cheap options for the web domain names, GoDaddy.com has a variety of Domain Names && hosting services.

* A free option (requires more work) is Github Pages. It is free and rather straight forward.

* Another great option with tons of products and services would be Linode.

* You must make sure the service comes with an SSL/TSL certificate, this will allow for the verification of the website with web browsers.

* HTML(Hyper Text Markup Language) is the framework for web pages, CSS (cascading style sheets) adds various style. This website is a simple markup using both.
