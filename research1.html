Research: Hosting a Website

Hosting vs Domain name

* In order to create a website, you need to purchase a Web Domain name (www.XYZ.com), as well as server space inorder to put the website online for others to view.

* There are very cheap options for the web domain names, GoDaddy.com has many options as well as hosting services.

* Another great option, that is even free (although, requires more work) is Github.io. They allow you to post a domain on their servers as well as give you server space. It is free!

* Another cheap option would be AWS. They have very cheap options that are indicative of traffic.

* You must make sure the service comes with an SSL/TSL certificate, this will allow for the verification of your site with hardware.
